## 主题

### 显示时间

```cmd
PROMPT $D$S$T$S$P$G
```

- `$D` 表示当前日期。
- `$S` 添加一个空格。
- `$T` 表示当前时间。
- `$S` 添加一个空格。
- `$P` 显示当前路径。
- `$G` 显示`>`字符。

在 CMD 中`PROMPT`环境变量用于定义命令行界面的提示符外观。通过更改`PROMPT`变量的值，用户可以自定义显示在每个命令行前的文本和信息。例如，可以设置它来显示当前目录、时间、日期等信息。
